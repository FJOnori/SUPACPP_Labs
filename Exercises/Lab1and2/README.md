
# User Instructions

- The "make" command compiles and runs the code.

- The "make clean" command removes any created files .txt files and the entire contents of the "build" folder.

- Compiled code is saved to "build".

- Input and output .txt files are saved to "data".

- Header files are saved to "include".

- C++ code is saved to "src".

- "make" runs the code in the main() function found in AnalyseData.cpp which performs task 11. This allows the user to input an integer of ethier 9,8,6,4 to run those respective tasks and save them to .txt files in the "data" folder. An input of "exit" kills the program and any other input throws up an error message without killing the program.
